Case Swapper Function

What is Case Swapper Function and How Do You Use It?

The Case Swapper Function is a utility that takes a string as input and swaps the case of each character within that string. For example, if you input the string 'The Quick Brown Fox', the output will be 'tHE qUICK bROWN fOX'. Simply type or paste your desired text into the prompt and hit OK. The function will then process your input string and display the result with the case of each character swapped.

Feel free to experiment with different inputs and enjoy the results!
